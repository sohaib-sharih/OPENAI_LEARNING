### OpenAI Learning Projects

This repository contains my hands-on experiments with the **OpenAI API** — covering text generation, image creation, audio processing, and building AI agents using Python. Each project folder (e.g., `setup_example_01`) represents a different learning chapter.

|Chapter|Topic|Description|
|---|---|---|
|1|**Setup Example 01**|Basic text generation using OpenAI API and Chainlit UI|
|2|File Search & Web Search|Exploring retrieval and live search tools|
|3|Image Generation|Creating visuals using OpenAI’s image models|
|4|Audio Tools|Converting text to speech and speech to text|
|5|Agents|Building interactive AI agents using OpenAI tools|
### About OpenAI

**OpenAI** provides a suite of advanced AI models that can understand and generate text, images, audio, and more. You can use these models through the **OpenAI Python SDK** or via API calls to integrate AI into your apps.

### Main OpenAI Models & Capabilities

|Model|Type|Capability|
|---|---|---|
|`gpt-5`, `gpt-4o`, `gpt-4o-mini`|Text → Text|Chat, reasoning, code, analysis|
|`gpt-4o-mini-tts`|Text → Speech|Converts written text into spoken audio|
|`gpt-4o-transcribe` / `gpt-4o-mini-transcribe`|Speech → Text|Transcribes audio into text|
|`gpt-4o-image`|Text → Image|Generates images from text prompts|
|`gpt-4o` (with tools)|Text + Tools|Supports web search, file search, and function calling|
